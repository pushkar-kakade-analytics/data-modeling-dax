# Data Modeling & DAX Patterns

## Purpose
This repository focuses on data modeling best practices and DAX calculation patterns used in business intelligence reporting.

## Coverage Areas
- Star schema and dimensional modeling
- Fact and dimension table design
- Time intelligence calculations
- Conditional and business-rule-driven DAX
- Optimization techniques for large datasets

## BI Context
The examples in this repository are aligned with real-world reporting requirements such as financial statements, KPIs, and management dashboards.

## Planned Additions
- YTD, MTD, and rolling metrics
- Complex conditional DAX logic
- Reusable calculation patterns
